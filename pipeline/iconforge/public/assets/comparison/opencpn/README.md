# OpenCPN Comparison Renders

These PNG files are OpenCPN/S-52 comparison evidence for public review.

- They are not Helm-owned canonical artwork.
- They are not part of the Apache-licensed Helm symbol asset pack.
- They are rendered comparison targets used to help reviewers spot shape,
  color, and semantic mismatches.
- Keep generated Helm SVG assets separate from this directory.

OpenCPN is GPL-2.0-or-later. Treat these rendered comparison PNGs as
GPL comparison artifacts unless counsel approves a narrower publication policy.
