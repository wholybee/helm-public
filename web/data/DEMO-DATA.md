# Demo data (synthetic, committed)

Everything here except `ais-sample.geojson`, `saved-sample.geojson`
is **synthetic demo data** generated by `pipeline/gen_demo_data.py` so the app runs
fully offline with no network and no GDAL/tippecanoe. **NOT FOR NAVIGATION.**
Real chart/depth overlays belong in `HELM_USER_DATA_ROOT`, `HELM_CONFIG/data`,
or `~/.helm/data`; the app serves that local directory at `/user-data/` and uses
it before these bundled demo fixtures.

Regenerate it all with one command:

```bash
python3 pipeline/gen_demo_data.py        # writes DEM, relief, contours, soundings,
                                         # AIS, radar, places, wind + PMTiles + GeoTIFF
```

| File / dir | What it is | Used by |
|---|---|---|
| `dem/{z}/{x}/{y}.png` | Terrarium-encoded DEM (bathymetry + land) | hillshade (Lab · mercator), value-encoding |
| `relief/{z}/{x}/{y}.png` | colour-relief raster | the PMTiles pack |
| `key-west-sat.pmtiles` | offline raster chart (hand-rolled PMTiles v3) | Lab · PMTiles |
| `key-west-depth.tif` | float32 depth GeoTIFF (EPSG:4326) | Lab · COG (`cog://`) |
| `depcnt.geojson` | depth contours, S-57 `VALDCO` | base chart `depcnt-line` |
| `soundg.geojson` | spot soundings, S-57 `DEPTH` | base chart `soundg-text` |
| `depth-contours.geojson` | same contours, `ele`/`depth_m`/`level` | Lab · Contours |
| `ais-fleet.geojson` | 448-vessel fleet (MMSI/SOG/COG/CPA) | Lab · AIS (deck.gl) |
| `radar/` + `manifest.json` | 6 precip frames | Lab · Temporal |
| `places.geojson` | marina/anchorage/fuel POIs (`kind`,`name`) | base chart `places-*` |
| `wind_points.geojson` | wind field (`dir_deg`,`speed_kt`) | base chart `wind-arrows` |
| `depare.geojson` | depth-area fill bands (`DRVAL1`) — run-length-merged grid | base chart `depare-fill` |

Why committed (vs. the `.gitignore` here that excludes pipeline output): this is
small, synthetic, and *not* license-bound, and the point is an instantly-runnable
demo. Real deployments keep license-bound chart/weather data out of git per the
existing rule; the synthetic files are force-added.
